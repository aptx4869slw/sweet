# 文件说明
## 输出运行目录
     + main/java                代码区域Restful风格，包含控制，业务，持久，主程序，定时器等
     + resources/static         静态资源块，包含JS,CSS,Image等
     + resources/mapper         映射文件数据持久化配置文件
     + resources/temolates      Web页面包含登陆，主页等文件
     + bin/application.yml      主配置文件
     + bin/license              软件license文件
     
### 主要包含
+ 登陆页面
+ 博客首页
+ 日志页面
+ 相册页面 
+ 弹幕留言页面
